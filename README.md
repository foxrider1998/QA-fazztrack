# QA-fazztrack
to run it just copy steps below  one by one 




``` 
npm install 
```


for running all test case use this
```
npm run wdio 
```

for running test case wajib
```
npm run wdio @wajib
```


for displaying reports, use
```
npm run allure-report 
```
